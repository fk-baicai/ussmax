分部签到开放窗口（北京时间）

使用说明

默认规则：未配置日期且未设长期默认时禁止签到。
勾选全天开放：当日随时可签；
不勾选：需设置开始时间与开放时长，仅规定时段内可签。
各分部「长期默认」：勾选启用后，下方日期格将锁定并统一沿用该规则；取消勾选后可逐日单独配置。
生效优先级：单日覆盖 > 长期默认 > 未配置（禁止签到）。启用长期默认保存时会清除该分部的单日覆盖，避免冲突。
时段签到时，OOPZ 在线时长仅统计签到开放窗口内的时间。
各分部签到积分可在分部标题旁单独设置，默认 1 分，签到成功自动加分。
修改某分部后请点击该分部标题行的「保存本分部」，仅保存当前分部，不会影响其他分部。

OOPZ 签到配置

说明

自动签到：已绑定 OOPZ 的成员，在 OOPZ 语音频道内按实际停留时长累计（机器人约每 60 秒上报一次）。达到分钟数后自动写入对应分部签到。「仅在签到时段内」勾选时，须该分部在上方开放窗口内才会自动签；时段签到下在线时长仅计开放窗口内。启用某分部自动签到时，若上方未配置该分部开放窗口，系统将自动设为「全天开放」。

手动签到：控制分部页「今日签到」按钮的额外条件。须绑定 OOPZ 时未绑定将引导去首页绑定；勾选「须在线时长」时，全天签到计今日总在线，时段签到仅计该分部开放窗口内在线（与自动签到共用统计）。

OOPZ 语音播报

说明

进语音频道时在 OOPZ 内 TTS 播报（非网站播放）。保存后约 30 秒同步；需启动 oopz-bot。

克隆音色（MOSS，拟真慢）与 edge-tts（进房约 1～3 秒，含普通话/方言女声）均可选。克隆参考音放 yise/；edge 可用 download_edge_voice_pack.py 探测本机可用项。

默认播报

同频道不同人之间最短间隔（默认 1 秒）。同一人 30 秒内重复进房不重复播（免冷却名单除外）。

免冷却名单

名单内 ID 每次进频道都播报（使用默认音色/文案，无需网站绑定），不受频道间隔与 30 秒内重复进房限制；同频道普通用户排队超过 5 人时合并欢迎，免冷却仍单独播报。请填写本人 OOPZ 数字 ID。

OOPZ ID 操作

成员定制播报

按 ID 单独设音色/文案；添加后无需网站绑定。留空则跟随默认。

密码留空则不修改。OOPZ ID 绑定后用户端不可自行更换或解绑，仅管理员可在此修改或解绑。仅超级管理员可改「fkbaicai」账户；超级管理员还可删除其他用户（不可删除超级管理员账户及本人）。

分部积分

使用说明

本表按总积分降序排列。
本月签到、积分、未签为北京时间当月汇总；后面日期为每日签到状态，√代表已签。
未签日期格可点击补签（仅限今日及以前；积分与当日实际签到一致，无记录时按分部当前设置）。
列表默认显示约 20 行，更多成员用右侧青蓝滚动条上下滑动；表头固定置顶。
表格过宽时，在空白处按住左键可左右拖拽平移。
横向滚动时「绑定 ID」与「调整值」两列始终钉在左侧；日期列有浅色虚线，鼠标悬停可高亮整列便于上下对照补签。
「下载当前月 xlsx」包含全部列（需先刷新）；下方可多选月份批量导出，每月单独一个工作表。

绑定 ID 调整值（可负） 邮箱 基础分 调整 总积分 本月签到（天） 本月签到积分 本月未签（天）

在「调整值」列（绑定 ID 右侧）：先填整数再点「应用」。上方搜索在已加载列表上即时筛选；下载 xlsx 仍为该分部全部成员（不受搜索影响）。

勾选月份后导出 xlsx，每月一表；月份多时点浅蓝条左右翻页。
